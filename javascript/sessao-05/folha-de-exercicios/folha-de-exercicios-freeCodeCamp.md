
# Gabarito – Exercícios JavaScript

**Questão 1**
C. Um erro será lançado porque const variáveis não podem ser reatribuídas.
→ Variáveis declaradas com const não podem ser reatribuídas após a inicialização.

**Questão 2**
B. const maxScore = 100;
→ O sinal de igual (=) é usado para atribuição em JavaScript.

**Questão 3**
B. Não, variáveis const devem ser inicializadas no momento da declaração.
→ Const exige valor inicial na declaração.

**Questão 4**
B. let
→ let permite declarar variáveis que podem ser atualizadas depois.

**Questão 5**
C. player1Score
→ Nomes de variáveis não podem começar com número, nem conter caracteres especiais ou palavras reservadas.

**Questão 6**
B. Nomes descritivos tornam seu código mais fácil de entender e manter.
→ Bons nomes facilitam a leitura e manutenção do código.

**Questão 7**
A. "Hello!"
→ Strings são valores entre aspas.

**Questão 8**
C. Boolean
→ Boolean representa true ou false.

**Questão 9**
C. Undefined
→ Variáveis declaradas sem valor têm tipo undefined.

**Questão 10**
C. O JavaScript fornece funcionalidade interativa e comportamento dinâmico para a página web.
→ JS adiciona interatividade e lógica às páginas.

**Questão 11**
C. O JavaScript interage com a página para alterar conteúdo e estilos dinamicamente.
→ JS pode modificar HTML e CSS em tempo real.

**Questão 12**
B. HTML e CSS são linguagens de marcação, enquanto JavaScript é uma linguagem de programação.
→ HTML/CSS estruturam e estilizam, JS programa.

**Questão 13**
B. 2
→ O operador de resto (%) retorna o valor que sobra da divisão. 5 % 3 resulta em 2, pois 3 cabe uma vez em 5 e sobra 2.

**Questão 14**
B. A divisão e a multiplicação acontecem antes da adição e subtração, assim como na matemática.
→ A precedência dos operadores segue as regras matemáticas.

**Questão 15**
B. Usando parênteses para forçar certas partes da expressão a serem avaliadas primeiro.
→ Parênteses alteram a ordem de avaliação.

**Questão 16**
C. Para representar valores verdadeiros ou falsos e tomar decisões baseadas em condições.
→ Booleanos são usados em decisões lógicas.

**Questão 17**
C. Porque ela verifica tanto o valor quanto o tipo, fornecendo resultados mais previsíveis.
→ === compara valor e tipo, evitando coerção automática.

**Questão 18**
B. Ela realiza a coerção de tipo (type coercion), convertendo os valores para o mesmo tipo antes de compará-los.
→ == converte tipos antes de comparar.

**Questão 19**
B. Ele verifica se o valor à esquerda é maior ou igual ao valor à direita.
→ >= retorna true se o valor à esquerda for maior ou igual ao da direita.
