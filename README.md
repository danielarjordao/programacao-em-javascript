# Programação em JavaScript – Programa Upskill ServiceNow

Material didático da disciplina de Programação em JavaScript, parte do Programa Upskill ServiceNow. Este repositório reúne exercícios resolvidos, exemplos práticos, anotações e pequenos projetos desenvolvidos ao longo do curso, com o objetivo de registrar o aprendizado, facilitar revisões e compartilhar conhecimento.

## Objetivo da Disciplina

Este repositório contém material introdutório para o ensino de:

- Fundamentos de desenvolvimento web (HTML, CSS e JavaScript)
- Estruturas de controle, funções, arrays e manipulação de dados em JavaScript
- Prática de lógica de programação e resolução de problemas
- Boas práticas de organização de código

## Como Usar Este Material

### Pré-requisitos

- Editor de código (recomendado: VS Code)
- Navegador web atualizado
- Conhecimentos básicos de terminal/linha de comando (opcional)

### Estrutura do repositório

- **fundamentos-web/**: Introdução ao desenvolvimento web, HTML, CSS e projetos iniciais.
- **javascript/**: Conteúdo focado em JavaScript, incluindo exemplos de código, folhas de exercícios e anotações das aulas.

### Executando os Exemplos

Abra os arquivos HTML em seu navegador para visualizar os projetos web.
Para os exemplos e exercícios em JavaScript, você pode:

- Executar diretamente no console do navegador
- Utilizar o Node.js para rodar arquivos `.js` (quando aplicável)

Exemplo:

```bash
node javascript/sessao-05/exemplos/script.js
```

## Principais Tópicos Abordados

- Sintaxe básica do JavaScript
- Operadores e estruturas de decisão (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`)
- Funções e escopo
- Arrays e métodos de array
- Manipulação de DOM (em projetos web)
- Prática com exercícios e pequenos projetos

> Este material é destinado ao estudo e registro de aprendizado na disciplina de Programação em JavaScript do Programa Upskill ServiceNow.
