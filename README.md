# Programação em JavaScript – Programa Upskill ServiceNow

Material didático da disciplina de Programação em JavaScript, parte do Programa Upskill ServiceNow. Este repositório reúne exercícios resolvidos, exemplos práticos, anotações e pequenos projetos desenvolvidos ao longo do curso, com o objetivo de registrar o aprendizado, facilitar revisões e compartilhar conhecimento.

## Objetivo da Disciplina

Ensinar os fundamentos e conceitos avançados de Programação em JavaScript, abrangendo:

- **Desenvolvimento Web**: HTML5, CSS3, DOM e eventos
- **Fundamentos de Linguagem**: Variáveis, operadores, arrays, loops, funções
- **Conceitos Avançados**: Assincronismo, Promises, Async/Await, Fetch API
- **Organização de Código**: Escopo, closures, modularização, boas práticas
- **Persistência**: Web Storage e manipulação de dados com JSON

Para uma referência rápida de todos os conceitos, consulte o [DICIONARIO.md](DICIONARIO.md).

## Como Usar Este Material

### Pré-requisitos

- Editor de código (recomendado: VS Code)
- Navegador web atualizado

- Conhecimentos básicos de terminal/linha de comando

## Principais Tópicos Abordados

### Fundamentos Web
- Comandos de terminal e Git basics
- HTML5 semântico e acessibilidade
- CSS3: Box Model, Flexbox, CSS Grid, Variáveis CSS
- Introdução ao DOM e manipulação de elementos

### JavaScript Essencial
- Tipos de dados (primitivos e por referência)
- Operadores e estruturas de controle (`if/else`, `switch`, ternário)
- Loops (`for`, `while`, `do...while`)
- Arrays e seus métodos (`map`, `filter`, `reduce`, `forEach`, etc.)
- Funções, closures e escopo
- Hoisting e Temporal Dead Zone (TDZ)

### JavaScript Avançado
- Funções de Alta Ordem (HOF)
- Destructuring e Spread/Rest operators
- Imutabilidade e Funções Puras
- **Assincronismo**: Callbacks, Promises, Async/Await
- **Fetch API** e requisições HTTP
- **Event Loop** e Call Stack
- **Tratamento de erros** em operações assíncronas

### DOM & Eventos
- Seleção de elementos (querySelector, getElementById, etc.)
- Manipulação de conteúdo e estilos
- Event listeners e propagação de eventos
- Event delegation
- Criação dinâmica de elementos

### Web Storage & Modularização
- `localStorage` e `sessionStorage`
- Persistência de estado com JSON
- **ES6 Modules**: Named e Default exports
- CommonJS vs ESModules
