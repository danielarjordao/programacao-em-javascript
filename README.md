# Programação em JavaScript - Programa Upskill ServiceNow

Este repositório reúne materiais, exercícios resolvidos, exemplos práticos e projetos desenvolvidos durante a disciplina de Programação em JavaScript do Programa Upskill, voltado para o programa ServiceNow. O objetivo é registrar o aprendizado, facilitar revisões e compartilhar conhecimento com outros participantes do programa.

## Estrutura do repositório

- **fundamentos-web/**: Introdução ao desenvolvimento web, HTML, CSS e projetos iniciais.
- **javascript/**: Conteúdo focado em JavaScript, incluindo exemplos de código, folhas de exercícios e anotações das aulas.
